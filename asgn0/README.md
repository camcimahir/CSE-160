# 2D Vector Math Visualizer

This is a simple sandbox application I built to test foundational 2D graphics concepts and get familiar with the HTML5 Canvas API before diving into more complex WebGL assignments. 

**Features:**
* **Interactive Canvas:** Draw and visualize two 2D vectors (V1 and V2) using X and Y coordinate inputs.
* **Vector Mathematics:** Performs and visualizes basic vector operations including Addition, Subtraction, Multiplication, Division, Magnitude, Normalization, Angle calculations, and Triangle Area calculations.
* **Visual Feedback:** Uses a color-coded system (Red for V1, Blue for V2, Green for Resulting Vectors) to easily verify the accuracy of the underlying matrix and vector math logic (`cuon-matrix-cse160.js`).
